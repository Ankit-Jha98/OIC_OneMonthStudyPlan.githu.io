🎯 Must-Know Topics for OIC Interviews
[01] XSLT Mapping — Every single OIC interview will ask you to write or explain an XSLT expression. concat(), if/choose, for-each are minimum.
[02] Error Handling — Global fault handler vs scope fault handler. How to retry. How to alert on failure. This separates beginners from real developers.
[03] Oracle ERP Adapter — Your secret weapon. You already know P2P, PO, iSupplier. Mapping that to OIC ERP adapter = instant credibility.
[04] Integration Patterns — App Driven vs Scheduled vs Event. When do you use which? Be ready with real examples.
[05] Your P2P Scenario — Build a story: "I integrated Oracle Fusion Procurement with [X] using REST adapter. Here is how I handled errors and mapped fields." Make it yours.
